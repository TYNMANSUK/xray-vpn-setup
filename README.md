# xray-vpn-setup

Автоматическая установка и настройка Xray + 3x-ui для Ubuntu VPS.

Скрипт выполняет полную подготовку сервера и автоматически подбирает рабочие связки (VLESS+Reality / Trojan+Reality) с учётом текущих блокировок РКН. После создания каждой связки проводится реальная проверка через временный Xray-клиент.

## Возможности

- Полная подготовка системы:
  - Создание swap (умный подбор под объём RAM)
  - Включение BBR + сетевые оптимизации
  - Установка и настройка fail2ban
  - Настройка ufw (firewall)
- Установка 3x-ui + Xray
- Автоматический подбор и проверка связок:
  - VLESS + Reality и Trojan + Reality
  - Разные fingerprint (приоритет: Firefox, Edge, Safari)
  - Разные порты и SNI
  - **Реальная проверка** — скрипт запускает Xray-клиент и тестирует подключение
- Удобное меню после установки

## Установка

### Быстрый способ

Выполните на чистом Ubuntu 22.04 / 24.04 от root:

```bash
curl -fsSL https://raw.githubusercontent.com/TYNMANSUK/xray-vpn-setup/main/install.sh | bash
```

### Ручной способ

```bash
wget https://raw.githubusercontent.com/TYNMANSUK/xray-vpn-setup/main/install.sh
chmod +x install.sh
sudo bash install.sh
```

## Использование

1. Запустите скрипт на новом VPS.
2. Дождитесь завершения установки.
3. В меню выберите пункт **1 — Автоматическая настройка**.
4. Скрипт сам переберёт варианты связок, проверит их и оставит только рабочие.
5. Все данные (логин/пароль панели + готовые ссылки) сохранятся в файл:

   `/root/xray-vpn-info.txt`

## Как работает проверка связок

Скрипт не просто создаёт инбаунды. Для каждой комбинации он делает следующее:

1. Создаёт инбаунд в панели 3x-ui.
2. Запускает временный Xray-клиент с теми же параметрами (publicKey, shortId, fingerprint, SNI и т.д.).
3. Поднимает локальный SOCKS-прокси.
4. Пытается выполнить реальный запрос (`curl` через прокси на google.com и 1.1.1.1).
5. Если запрос успешно прошёл — связка считается рабочей.

Только проверенные связки попадают в итоговый список.

## Файлы после установки

| Файл                              | Описание                          |
|-----------------------------------|-----------------------------------|
| `/root/xray-vpn-info.txt`         | Данные панели и готовые ссылки    |
| `/var/log/xray-vpn-install.log`   | Полный лог установки              |
| `/etc/sysctl.d/99-xray-vpn.conf`  | Сетевые оптимизации               |

## Требования

- Ubuntu 22.04 LTS или 24.04 LTS
- Права root
- Чистый сервер (рекомендуется)

## После установки

- Зайдите в панель по адресу, который выдал скрипт
- Сразу смените пароль администратора
- При необходимости добавьте клиентов

## Удаление

```bash
x-ui uninstall
rm -f /swapfile
sed -i '/swapfile/d' /etc/fstab
```
