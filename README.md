# Автоматическая настройка Xray VPN для Ubuntu

Скрипт для быстрой автоматической установки и настройки Xray + 3x-ui на Ubuntu VPS.

После установки скрипт сам подбирает и проверяет рабочие связки (VLESS + Reality / Trojan + Reality) с учётом текущих блокировок РКН.

## Возможности

- Полная подготовка системы (swap, BBR, fail2ban, ufw)
- Установка 3x-ui + Xray
- Автоматический подбор связок:
  - VLESS + Reality и Trojan + Reality
  - Разные fingerprint (в приоритете Firefox, Edge, Safari)
  - Разные порты и SNI
- **Реальная проверка** связки — скрипт запускает временный Xray-клиент и тестирует соединение
- Меню после установки:
  1. Автоматическая настройка (рекомендуется)
  2. Показать данные для входа в панель
  3. Выход

## Установка

### Быстрый способ (рекомендуется)

```bash
curl -fsSL https://raw.githubusercontent.com/ВАШ_ЮЗЕР/ВАШ_РЕПО/main/install.sh | bash
```

### Ручная установка

```bash
wget https://raw.githubusercontent.com/ВАШ_ЮЗЕР/ВАШ_РЕПО/main/install.sh
chmod +x install.sh
sudo bash install.sh
```

**Важно:** замените `ВАШ_ЮЗЕР/ВАШ_РЕПО` на ваш репозиторий на GitHub.

## Использование

1. Запустите скрипт на чистом Ubuntu 22.04 или 24.04 (от root).
2. Дождитесь завершения установки.
3. В меню выберите **1** — Автоматическая настройка.
4. Скрипт сам попробует разные варианты и проверит, какие связки реально работают.
5. В конце все рабочие ссылки будут сохранены в файл `/root/xray-vpn-info.txt`.

## Как работает проверка связок

Скрипт не просто создаёт инбаунды. Для каждой связки он:

1. Создаёт инбаунд в панели.
2. Запускает временный Xray-клиент с теми же параметрами (fingerprint, shortId, publicKey и т.д.).
3. Пытается сделать реальный запрос через прокси (`curl` на google.com и 1.1.1.1).
4. Если запрос прошёл успешно — связка считается рабочей.

Это позволяет отсеивать связки, которые не работают даже при правильных настройках.

## Созданные файлы

- `/root/xray-vpn-info.txt` — данные панели и готовые ссылки
- `/var/log/xray-vpn-install.log` — полный лог установки
- `/etc/sysctl.d/99-xray-vpn.conf` — сетевые оптимизации

## Требования

- Ubuntu 22.04 LTS или 24.04 LTS
- Права root
- Чистый сервер (рекомендуется)

## После установки

- Зайдите в панель по адресу, который выдал скрипт
- Смените пароль администратора
- При необходимости создайте дополнительные инбаунды вручную

## Удаление

```bash
x-ui uninstall
rm -f /swapfile
sed -i '/swapfile/d' /etc/fstab
```
